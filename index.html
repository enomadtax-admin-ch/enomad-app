🗄 Supabase — Database permanente
Come configurare
1. Vai su supabase.com e crea un progetto gratuito
2. Copia la Project URL e la anon key dalle impostazioni API
3. Incollale qui sotto e salva — i dati si sincronizzeranno automaticamente
Aggiornamenti app futuri:
Con Supabase attivo puoi sostituire l'HTML su GitHub in qualsiasi momento. L'app leggerà i tuoi dati dal DB senza toccarli. Zero rischi.
📦 Backup JSON
Export — scarica i tuoi dati
Esporta tutti i dati in un file .json — leggibile, importabile, editabile. Usalo come backup manuale prima di ogni aggiornamento dell'app.
Import — carica dati da file
Importa un backup precedente. I dati esistenti verranno sostituiti. Fai sempre un export prima.
